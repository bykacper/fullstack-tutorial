Ten paragraf ma nadane unikalne id. Służy ono do manipulowania elementem poprzez np. CSS czy JavaScript. Każde id powinno być niepowtarzające się, unikalne, aby uniknąć potencjalnych błędów. Tylko jeden element może mieć jakieś id.

Ten paragraf ma nadaną klasę. Atrybut ten pomaga grupować i stylizować wiele elementów w ten sam sposób za pomocą CSS, lub manipulować nimi za pomocą JavaScript. Elementów o tej samej klasie może być wiele.
